
# 🤖 AI Chat App — Flask + Groq AI

A full-stack AI chat web application built with Flask (backend) 
and HTML/CSS/JS (frontend), powered by Groq's free LLaMA 3.3 70B API.

## 🚀 Features
- 💬 Real-time chat with Llama 3.3 70B AI model
- 🧠 Full conversation memory across turns
- ✨ Markdown rendering (code blocks, bold, lists)
- 📜 Chat history in sidebar
- 🔄 New Chat / Clear chat functionality
- 💡 Quick suggestion prompts
- 📱 Responsive dark-themed UI
- ⚠️ Error handling with toast notifications

## 🛠 Tech Stack
| Layer    | Technology                  |
|----------|-----------------------------|
| Frontend | HTML5, CSS3, Vanilla JS     |
| Backend  | Python 3, Flask             |
| AI Model | Groq API (Llama 3.3 70B)    |
| Styling  | Custom CSS (dark theme)     |

## ⚡ Setup
1. Clone the repo
2. Create virtual environment and activate it
3. Run `pip install -r requirements.txt`
4. Copy `.env.example` → `.env` and add your Groq API key
5. Run `python app.py`
6. Open http://localhost:5000

## 🔑 Get a Free Groq API Key
Visit https://console.groq.com → Sign up → API Keys → Create Key
